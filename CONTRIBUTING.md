# Contributing to SmartJunction AI

We're excited you want to help! Here’s how you can contribute:

## 🚀 How to Help

1.  **Report Bugs**: Open an issue describing the bug and how to reproduce it.
2.  **Suggest Features**: Want to see more sensors? Better vision? Open an issue!
3.  **Submit Pull Requests**: 
    - Fork the repo.
    - Create a branch (`feature/amazing-new-logic`).
    - Commit your changes.
    - Push and open a PR.

## 📝 Coding Standards

- **Arduino**: Use descriptive variable names and comment complex logic.
- **Node-RED**: Keep your flows organized and use descriptive names for function nodes.
- **Documentation**: Update the README if you change hardware pins or software logic.
